# Release v0.1.36

## Neue Features
- Integration eines MediathekViewWeb-Feed-Fallbacks in der Serien-Suche für zuverlässigere Suchergebnisse

## Verbesserungen
- Optimierte API-Abfrage-Strategie:
  - Erste Anfrage mit minimalem Query für schnelle Basis-Ergebnisse
  - Nachfolgende Anfragen mit erweiterten Feldern für detailliertere Ergebnisse

## Technische Änderungen
- Übersetzung der Log-Handler-Kommentare und Docstrings ins Deutsche in der GUI-Komponente

_Hinweis: Dieses Release enthält keine Bugfixes._
