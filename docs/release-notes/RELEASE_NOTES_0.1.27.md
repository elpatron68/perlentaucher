# Release v0.1.27

### Neue Features
- Doppelklick auf Feed-Einträge öffnet den zugehörigen Blog-Post im Browser
- Integration von App-Icons für die Anwendung
- Logo im About-Dialog hinzugefügt

### Verbesserungen
- Dokumentation auf informellere Ansprache ("Du" statt "Sie") umgestellt
- Versionshinweise in der GUI-Dokumentation neutraler formuliert
- Begrifflichkeit von "Blog-Eintrag" zu "Blog-Post" in Benachrichtigungen angepasst

### Bugfixes
- Problem mit doppelten Asset-Uploads bei Codeberg Releases behoben

### Technische Änderungen
- PyInstaller-Unterstützung durch resource_path() Hilfsfunktion implementiert
- Icon-Support für PyInstaller-Executables eingerichtet
