# Release v0.1.33

### Verbesserungen
- Optimiertes Import-Handling für PyInstaller auf allen Plattformen (Windows/Linux/macOS)

### Bugfixes
- Behebung von GUI-Startproblemen unter Linux durch angepasste Import-Struktur

### Technische Änderungen
- Umstellung auf relative Imports in GUI-Modulen für PyInstaller-Kompatibilität
- Anpassung des GUI-Import-Fallbacks für PyInstaller unter Linux

Die Hauptänderungen in diesem Release konzentrieren sich auf die Verbesserung der PyInstaller-Kompatibilität, insbesondere im Hinblick auf plattformübergreifende Stabilität und GUI-Funktionalität unter Linux.
