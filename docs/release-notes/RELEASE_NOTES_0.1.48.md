# Release v0.1.48

## Neue Features
- **Schalter -SkipRelease für GitHub/Codeberg-Releases**  
  - Hinzugefügt in `release.ps1`, ermöglicht das Erstellen von Releases ohne die Notwendigkeit für einen vollständigen Release-Prozess.  
  Co-authored-by: Cursor <cursoragent@cursor.com>

## Verbesserungen
- **Versionierung im CHANGELOG**  
  - Die Struktur des CHANGELOG wurde aktualisiert, um die Unreleased-Versionen mit dem Tag `-SkipRelease` zu kennzeichnen.  
  Co-authored-by: Cursor <cursoragent@cursor.com>

## Technische Änderungen
- **Einrückung in SkipRelease-Zweigen vereinheitlicht**  
  - Anpassungen in `release.ps1`, um die Konsistenz der Einrückungen in den SkipRelease-Zweigen sicherzustellen.  
  Co-authored-by: Cursor <cursoragent@cursor.com>
