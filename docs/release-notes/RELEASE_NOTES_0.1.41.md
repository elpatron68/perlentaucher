# Release v0.1.41

## Verbesserungen
- **Seriendownload**: Verbesserte Behandlung von Umlauten
  - Deutsche Umlaute (ä, ö, ü, ß) bleiben bei der Suche erhalten
  - Keine automatische Konvertierung mehr in ae/oe/ue

## Bugfixes
- **Seriendownload**: Implementierung eines Fallbacks für Episoden ohne Staffel/Episoden-Information

Die Änderungen in diesem Release verbessern hauptsächlich die Handhabung deutscher Zeichen und die Robustheit bei fehlenden Metadaten.
