# Release v0.1.38

### Technische Änderungen
- **Build-Prozess macOS**
  - Verbessertes Build-Script für GUI unter macOS
  - Nutzung von virtualenv (.venv) wenn verfügbar
  - Optimierung des PyInstaller build.spec (ohne universal2)
  - Vereinfachung des macOS-Workflows
  - Bereinigter Build-Prozess durch --clean Flag

### Zusammenfassung
Dieses Release konzentriert sich auf technische Verbesserungen des Build-Prozesses für macOS, mit Fokus auf bessere Kompatibilität und vereinfachte Build-Abläufe.
